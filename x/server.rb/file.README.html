About

server.rb is a static file web server implemented with puma and rack.

Example

Server.for

The Server.for method receives the path to a directory, and returns an instance of Server. The Server#start method can spawn a HTTP server on a separate thread, and the thread can be blocking or non-blocking:

require "server"
server = Server.for File.join(Dir.getwd, "website")
server.start(block: true)

Options

Server.for accepts options that are forwarded onto the Puma web server. The unix, host, and port options are supported by server.rb and translated to a matching binds option that Puma natively supports:

require "server"
server = Server.for File.join(Dir.getwd, "website"),
                    unix: File.join(Dir.getwd, "website.sock")

Documentation

A complete API reference is available at 0x1eef.github.io/x/server.rb

Install

server.rb can be installed via rubygems.org:

gem install server.rb
