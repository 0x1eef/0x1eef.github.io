Overview

The LLM::Anthropic::Models class provides a model object for interacting with Anthropic's models API. The models API allows a client to query Anthropic for a list of models that are available for use with the Anthropic API.

Examples:

#!/usr/bin/env ruby
require "llm"

llm = LLM.anthropic(key: ENV["KEY"])
res = llm.models.all
res.each do |model|
  print "id: ", model.id, "\n"
end

def all(**params)
  query = URI.encode_www_form(params)
  req = Net::HTTP::Get.new("/v1/models?#{query}", headers)
  res, span = execute(request: req, operation: "request")
  res = ResponseAdapter.adapt(res, type: :enumerable)
  finish_trace(operation: "request", res:, span:)
end
