Overview

The LLM::ServerTool class represents a platform-native tool that can be activated by an LLM provider. Unlike LLM::Function, these tools are pre-defined by the provider and their capabilities are already known to the underlying LLM.

Examples:

#!/usr/bin/env ruby
llm = LLM.gemini ENV["KEY"]
ses = LLM::Session.new(llm, tools: [LLM::ServerTool.new(:google_search)])
ses.talk("Summarize today's news", role: :user)
print ses.messages.find(&:assistant?).content, "\n"

def to_h
  case provider.class.to_s
  when "LLM::Anthropic" then options.merge("name" => name.to_s)
  when "LLM::Gemini" then {name => options}
  else options.merge("type" => name.to_s)
  end
end
