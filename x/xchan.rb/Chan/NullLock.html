Overview

Chan::NullLock is a no-op lock that can be used instead of the standard file lock when a lock is not needed

Examples:

ch = xchan(:marshal, lock: Chan::NullLock)
ch.send([1,2,3])
# ditto
ch = xchan(:marshal, lock: :null)
ch.send([1,2,3])
