Overview

Ryo::Object is a Ryo object and subclass of Ruby’s Object class that can be created by using Ryo(), Ryo.Object(), Ryo::Object.from, or Ryo::Object.create.

def initialize_dup(ryo)

  Ryo.extend!(self, Ryo)
